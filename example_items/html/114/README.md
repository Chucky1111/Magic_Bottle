# 万分寻常之瓶 - 玻璃球游戏

一个体现魔瓶特质（编号114，生活系）的简单网页游戏。没什么特别的，就是几个玻璃球在画布上碰撞，触发些寻常的情绪效果。

## 设计思路

### 魔瓶特质融入
- **核心驱动力**："七情六欲皆伤身，开心悲伤都无趣，打油一首躺椅睡"
  - 游戏避免强烈情绪刺激，所有效果都平淡温和
  - 碰撞效果轻微，颜色变化柔和
  - 没有得分、没有胜利条件、没有失败惩罚

- **行为惯性**："躺躺椅，摇摇扇，日上杆头七八分，睡到雁南回北时"
  - 界面右下角有个躺椅图标，缓慢摇摆
  - 右上角有个风扇图标，缓慢旋转
  - 整体节奏慵懒，动画缓慢

- **世界观基石**："暴风烈日不常有，常常都是寻常阴晴日"
  - 背景采用柔和的灰白色渐变
  - 没有强烈对比色
  - 网格背景象征日常的规律性

### 情绪魔药体现
四个情绪魔药以平淡方式呈现：
1. **初生的惊喜** - 淡黄色效果，短暂闪烁
2. **如火焰一般追逐闪电** - 橙色效果，稍显活跃但仍克制
3. **轻蔑的嘲弄** - 紫色效果，略带讽刺意味
4. **竞技场的散场时刻** - 褐色效果，终结感但不沉重

情绪触发时，整个背景会有轻微颜色变化，但很快回归寻常。

## 技术实现

### 项目结构
```
├── index.html          # 主页面
├── src/
│   ├── main.ts        # 游戏主逻辑
│   └── style.css      # 样式文件
├── package.json       # Bun包管理配置
├── tsconfig.json      # TypeScript配置
├── vite.config.ts     # Vite开发服务器配置
└── README.md          # 本文件
```

### 主要特性
1. **Canvas物理引擎**
   - 简单的碰撞检测和响应
   - 动量守恒模拟
   - 边界反弹

2. **拖拽交互**
   - 鼠标和触摸屏支持
   - 拖动时球体变色
   - 释放后继续物理运动

3. **情绪效果系统**
   - 碰撞随机触发情绪
   - 颜色渐变过渡
   - 短暂的情绪状态显示

4. **响应式设计**
   - 适应不同屏幕尺寸
   - 移动端触摸优化

### 开发命令
```bash
# 安装依赖
bun install

# 启动开发服务器
bun run dev

# 构建生产版本
bun run build

# 预览构建结果
bun run preview
```

## 风格选择

### 视觉风格
- **颜色方案**：以灰、白、蓝为主，搭配情绪色作为点缀
- **字体**：使用"马善政"体体现中文书法韵味，"Noto Sans SC"确保可读性
- **图标**：Font Awesome图标，简洁明了
- **布局**：左侧游戏区域，右侧信息面板，符合寻常浏览习惯

### 交互风格
- **缓慢动画**：所有过渡效果都使用较慢的时长
- **轻微反馈**：交互反馈克制，避免过度反应
- **无紧迫感**：没有时间限制，没有压力元素

### 代码风格
- **TypeScript**：类型安全但不过度复杂
- **模块化**：逻辑清晰但不过度设计
- **注释**：必要处添加，不冗余

## 使用说明

1. 打开 `index.html` 或访问开发服务器
2. 拖拽球体让它们互相碰撞
3. 观察情绪变化（但别太在意）
4. 点击"加个球"添加新球
5. 点击"清空"重新开始
6. 点击右侧情绪药水手动触发效果

## 设计哲学

整个游戏试图体现"寻常"的美学：
- 不过度设计
- 不追求刺激
- 不制造焦虑
- 只是存在，只是发生
