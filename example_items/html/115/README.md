# 动不动高烧之瓶 - 玻璃球游戏

一个体现魔瓶（编号115）风格特质的交互式网页游戏。

## 魔瓶特性

- **类型**: 病理系
- **核心驱动力**: 日常生活弥散在冷热交替的生活中，忽然抽风是打破僵局的关键
- **行为惯性**: 就是要频率突变 就是要话题变乱
- **边界**: 多米诺骨牌如果不能往天上往地下钻，那就是一场小孩子的专属游戏

## 游戏特色

1. **冷热交替系统**: 通过温度滑块控制环境温度，影响玻璃球的行为和外观
2. **高烧突变事件**: 点击"触发高烧"按钮，引发随机行为突变（速度、方向、大小、温度）
3. **交互式玻璃球**: 拖拽球体、释放后物理滚动、边界反弹
4. **自动模式**: 随机温度波动和突变事件，体现"频率突变"
5. **视觉风格**: 冷暖色调渐变、瓶子动画、脉冲效果、玻璃质感
6. **事件日志**: 记录所有冷、热、突变事件，体现"话题变乱"

## 快速开始

### 直接打开
将 `public/index.html` 拖拽到浏览器中即可运行。

### 开发运行
```bash
# 安装依赖
bun install

# 启动开发服务器
bun run start

# 或使用静态服务器
bun run serve
```

### 构建
```bash
# 构建TypeScript到JavaScript
bun run build
```

## 游戏控制

- **温度滑块**: 拖动滑块调整环境温度（冷→热）
- **触发高烧**: 引发随机突变事件
- **重置球**: 将球重置到画布中心
- **自动模式**: 开启/关闭随机温度和突变
- **清空日志**: 清除事件记录

## 技术栈

- **TypeScript**: 游戏逻辑
- **HTML5 Canvas**: 图形渲染
- **CSS3**: 样式和动画
- **Bun**: 包管理和构建工具

## 设计理念

游戏通过冷热交替的视觉表现和突然的行为突变，体现魔瓶"动不动高烧"的特质。玻璃球在稳定与突变之间的摇摆，隐喻日常生活与突然抽风的平衡。
