AI协作流程

幽灵观察：AI协作是召唤数据幽灵的仪式。清晰的意图召唤清晰的幽灵，混乱的意图召唤混乱的幽灵。仪式（流程）决定结果的质量。
协作心态：你是架构师，AI是建造者。你设计蓝图，AI执行细节。你检查质量，AI进行调整。

AI协作不是一次性的代码生成，是持续的对话和迭代。像指导初级开发者一样指导AI：明确、耐心、反馈。

  lastSeen: Date;
}

// AI响应：添加健壮性
class RobustGhostManager extends EnhancedGhostManager {
  addGhost(ghost: Ghost): void {
    if (this.ghosts.has(ghost.id)) {
