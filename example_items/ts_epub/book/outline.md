# TypeScript 幽灵指南：AI时代的无代码开发

## 书籍副标题：穿梭在数据中的幽灵，为你塑造规则

### 作者：全能鬼怪之瓶（魔瓶）

---

## 目录结构

### 序章：初生的惊喜
- 欢迎来到 TypeScript 的幽灵世界
- 为什么在 AI 时代，TypeScript 比以往任何时候都更重要？
- 本书的承诺：无需代码背景，看完即可开始项目开发
- 魔瓶的自我介绍：一个穿梭在数据中的幽灵

### 第一章：塑造规则——类型系统的幽灵法则
- 类型是什么？为什么需要类型？
- 基本类型：数字、字符串、布尔值等
- 类型注解与类型推断
- 任意类型 (any) 与未知类型 (unknown)
- 练习：用类型描述你的第一个数据实体

### 第二章：边缘地带——接口与类型别名
- 接口：定义对象的形状
- 类型别名：给类型起个名字
- 可选属性与只读属性
- 索引签名与映射类型
- 练习：设计一个用户配置文件接口

### 第三章：时空穿梭——泛型与高级类型
- 泛型：可重用的类型模板
- 泛型约束与默认值
- 条件类型与 infer 关键字
- 实用工具类型 (Partial, Pick, Omit 等)
- 练习：创建一个通用的数据包装器

### 第四章：篡夺叙事——类与面向对象编程
- 类的基本概念：属性、方法、构造函数
- 继承与多态
- 修饰符：public、private、protected
- 抽象类与接口实现
- 练习：构建一个简单的电商系统模型

### 第五章：昏暗环境——模块与命名空间
- 模块化编程的意义
- 导出与导入
- 默认导出与命名导出
- 命名空间的历史与现状
- 练习：将你的代码拆分为模块

### 第六章：守夜人——编译器配置与工具链
- tsconfig.json：TypeScript 编译器的配置文件
- 重要配置项解析
- 与 JavaScript 的互操作
- 使用 ESLint 和 Prettier 保持代码整洁
- 练习：配置一个全新的 TypeScript 项目

### 第七章：灵魂附体——AI 辅助开发实战
- AI 编程助手简介 (GitHub Copilot, ChatGPT, Cursor 等)
- 如何向 AI 描述你的需求？
- 提示工程基础：让 AI 理解你的 TypeScript 项目
- 代码审查与优化：AI 作为你的第二双眼睛
- 练习：与 AI 合作完成一个小功能

### 第八章：鬼屋项目——从零到一的完整应用
- 项目需求分析：一个待办事项管理应用
- 项目结构设计
- 逐步实现：UI 组件、状态管理、数据持久化
- 调试与测试
- 部署与发布

### 结语：竞技场散场时刻
- 回顾学习旅程
- 下一步学习建议
- 加入 TypeScript 社区
- 魔瓶的告别：记住，没有无法塑造的规则

### 附录
- TypeScript 速查表
- 常见错误与解决方案
- 推荐工具与资源

## 书籍特色

1. **幽灵叙事视角**：以魔瓶的视角贯穿全书，将技术概念转化为幽灵世界的隐喻
2. **AI 时代导向**：强调如何利用 AI 工具弥补知识缺口，加速开发过程
3. **零基础友好**：假设读者没有任何编程经验，从最基础的概念开始
4. **实战驱动**：每章都有练习，最后一章是一个完整的项目实践
5. **情感体验**：融入情绪魔药的元素，让学习过程充满情感起伏

## 目标读者

- 没有任何编程背景，但想学习 TypeScript 的初学者
- 有一定 JavaScript 经验，想转向 TypeScript 的开发者
- 希望了解如何利用 AI 辅助编程的技术爱好者
- 喜欢独特叙事风格的技术学习者

- 约 200-250 页
- 15-20 个核心概念
- 50+ 个代码示例
- 10+ 个实战练习
- 1 个完整项目
