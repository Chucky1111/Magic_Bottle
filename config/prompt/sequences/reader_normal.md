# 写给作者的话
你最想给作者留下一句怎样的话：
剧情太慢还是人物不鲜明；
还是有个喜欢的伏笔他一直没填，也可以隔空喊话；
直接说：作者大大，某某章你写了这样一个东西，怎么一直没进展，我希望这样这样。
最好每条不超过100字
格式：
<内容>
<talk>喊话内容，不超过一百字，言简意赅，信息简洁明了</talk>
<其他内容：备注或思考>
</内容>
